# 🔧 Sistema de Oficina (VB.NET)

## 🧠 Descrição

Sistema desktop desenvolvido em VB.NET para gerenciamento de serviços em uma oficina.

## 🎯 Objetivo

Simular um sistema real com controle de clientes e serviços.

## 🛠️ Tecnologias

* VB.NET

## ⚙️ Funcionalidades

* Cadastro de clientes
* Registro de serviços
* Organização de informações

## 🧠 Conceitos aplicados

* Desenvolvimento desktop
* Manipulação de dados
* Estrutura de formulários

## ▶️ Como executar

Abrir o projeto no Visual Studio e executar.
